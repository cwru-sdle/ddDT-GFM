ddDT-GFM
========

Data-driven Foundation Models for fleet-level system performance prediction,
advanced manufacturing tool and part qualification, conformance, and lifetime.

The package uses the flexibility of spatiotemporal graph neural networks
(st-GNNs) to apply a single self-supervised methodology across conceptually
different systems: solar photovoltaic fleets, direct ink write additive
manufacturing, and laser powder bed fusion.
